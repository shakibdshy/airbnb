# Airbnb Clone with Nextjs, Nodejs, Mongoose & TypeScript

* Frontend
  - TypeScript
  - Nextjs
  - MUI (Material UI)
* Backend
  - Nodejs
  - Express.js & Express.js middleware
  - MongoDB with Mongoose
  - TypeScript
  - REST API principals

    - HTTP methods
  * JWT & refresh tokens
