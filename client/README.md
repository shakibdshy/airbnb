# Airbnb Clone with Nextjs, Nodejs, Mongoose & TypeScript
This is a Airbnb Clone.

# Technologies

* Frontend
  - TypeScript
  - Nextjs
  - MUI (Material UI)
* Backend
  - Nodejs
  - Express.js & Express.js middleware
  - MongoDB with Mongoose
  - TypeScript
  - REST API principals
    - CRUD
    - HTTP methods
  * JWT & refresh tokens
